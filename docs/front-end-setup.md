# Frontend Setup

## Prerequisites
- Node.js (v18+ recommended)
- npm (v9+ recommended)

## Install dependencies

```
cd src/frontend
npm install
```

## Run the development server

```
npm run dev
```

The frontend will be available at http://localhost:5173 by default.

## Build for production

```
npm run build
```

## Lint and format

```
npm run lint
npm run format
```
